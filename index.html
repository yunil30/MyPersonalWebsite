My Portfolio

YouTube Clone

YouTube Clone is a web application inspired by the original YouTube website. It has a similar user interface as the original YouTube website but was limited to 12 YouTube videos that were linked to the original videos using the Anchor tag. I built it using HTML and CSS.

My Activity List

My Activity List is a ToDo App web application, that allows a user to create their own personal activity list. It can add an activity by clicking the Add Activity button and can also remove the activity when it is finished by clicking the Delete button. I built it using HTML, CSS and Javascript.
